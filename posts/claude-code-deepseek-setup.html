前置检查

在开始之前，确认你的 Windows 版本为 Windows 10 及以上。以下所有操作均在 PowerShell 终端中执行。

打开 PowerShell 的方法：按 Win + X，选择 Windows PowerShell（不需要管理员权限）。

Step 1：环境准备 — 安装 Node.js

1.1 检查是否已安装

PS> node --version

如果有输出版本号（如 v22.14.0），说明已有 Node.js，跳到 Step 2。
如果提示"无法识别"，则继续安装。

1.2 方式 A（推荐）：winget 安装

PS> winget install OpenJS.NodeJS.LTS
正在下载并安装... 完成后重启终端即可

1.3 方式 B：下载安装包

打开 https://nodejs.org/，点击左侧 LTS 版下载 .msi 安装包，运行后一路 Next。

注意：安装过程中勾选 "Automatically install the necessary tools"，这样会自动配置 PATH 环境变量。

1.4 验证安装

PS> node --version
v22.14.0
PS> npm --version
10.x.x
node 命令找不到？ 重启 PowerShell 再试。如果还不行，重启电脑（安装包已将 Node.js 加入系统 PATH，但需要新会话才能加载）。

版本要求：Node.js v18.0.0 以上，推荐 v22 LTS。低于 v16 的版本安装 Claude Code 会报错。

Step 2：安装 Claude Code

确保 Node.js 安装成功后，在 PowerShell 中运行：

# 使用国内镜像加速（国内用户一定要加）
PS> npm install -g @anthropic-ai/claude-code --registry https://registry.npmmirror.com
added 1 package in 15s
卡在 Installing？ 确认加了 --registry https://registry.npmmirror.com。国内连 npm 官方源极慢甚至超时，镜像源能提速 100 倍。如果实在不行，可以永久切换镜像源：
npm config set registry https://registry.npmmirror.com

验证安装：

PS> claude --version
2.1.157

如果报"无法加载文件"之类的策略错误，先解除 PowerShell 的执行策略限制：

PS> Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
# 输入 Y 确认即可
Windows 安全提示？ 可能会弹出 Windows Defender 阻止运行的窗口，点击"仍然运行"即可。这是 npm 全局安装触发的正常行为，不是病毒。

Step 3：获取 DeepSeek API Key

Claude Code 默认连接 Anthropic 官方 API，国内无法访问。我们需要把它指向 DeepSeek 的兼容接口。

  1. 打开 https://platform.deepseek.com/，注册或登录账号
  2. 左侧菜单点击 API Keys
  3. 点击 创建新的 API Key
  4. 复制生成的 sk-xxxxxxxx 字符串
Key 务必保密！不要发到公开网络、不要提交到 GitHub。如果泄露，去 DeepSeek 平台删除并重新生成。
需要充值吗？ DeepSeek V4 Pro 需要账户有余额才能调用。充值入口在平台右上角，最低充值金额通常为 10 元。最新价格：输入（缓存命中）每百万 tokens 仅 0.025 元，输入（缓存未命中）3 元，输出 6 元。关键是这个价格已经永久降价了，不是限时折扣，长期用完全不慌。
喜欢其他模型？一样的方法。 如果你平时用 Kimi、GLM、通义千问或其他国产大模型，也可以用同样的配置方式接入 Claude Code。只需要把 ANTHROPIC_BASE_URL 换成对应模型厂商提供的 Anthropic 兼容接口地址，ANTHROPIC_AUTH_TOKEN 换成对应的 API Key 即可。原理都一样，换个端点的事。

Step 4：创建配置文件 settings.json

在用户目录下创建 .claude 文件夹，在里面放配置文件：

C:\Users\你的用户名\
  |
  +-- .claude\ <-- 新建这个文件夹
  |     |
  |     +-- settings.json <-- 新建这个文件

4.1 方法一：PowerShell 命令创建

PS> mkdir $env:USERPROFILE\.claude
PS> notepad $env:USERPROFILE\.claude\settings.json

4.2 方法二：图形界面创建

按 Win + R 打开运行，输入 %userprofile%\.claude 回车。在打开的文件夹里右键，新建 settings.json 文件。

文件创建后，完整复制粘贴以下内容，把 sk-你的Key 替换成上一步拿到的 API Key：

{
  "env": {
    // DeepSeek API Key，从 platform.deepseek.com 获取
    "ANTHROPIC_AUTH_TOKEN": "sk-你的Key",
    
    // DeepSeek 提供的 Anthropic 兼容接口地址（国内直连）
    "ANTHROPIC_BASE_URL": "https://api.deepseek.com/anthropic",
    
    // 默认模型
    "ANTHROPIC_MODEL": "deepseek-v4-pro[1m]",
    
    // Opus 档 -> V4 Pro（最强，复杂任务）
    "ANTHROPIC_DEFAULT_OPUS_MODEL": "deepseek-v4-pro[1m]",
    
    // Sonnet 档 -> V4 Pro（主力，日常开发）
    "ANTHROPIC_DEFAULT_SONNET_MODEL": "deepseek-v4-pro[1m]",
    
    // Haiku 档 -> V4 Flash（轻量，快速响应）
    "ANTHROPIC_DEFAULT_HAIKU_MODEL": "deepseek-v4-flash",
    
    // 禁用不必要的网络请求（保护隐私）
    "CLAUDE_CODE_DISABLE_NONESSENTIAL_TRAFFIC": "1",
    
    // 最高努力程度，深度推理
    "CLAUDE_CODE_EFFORT_LEVEL": "max"
  }
}

也可以复制以下纯文本版本（不带注释，更干净）：

{"env":{"ANTHROPIC_AUTH_TOKEN":"sk-你的Key","ANTHROPIC_BASE_URL":"https://api.deepseek.com/anthropic","ANTHROPIC_MODEL":"deepseek-v4-pro[1m]","ANTHROPIC_DEFAULT_OPUS_MODEL":"deepseek-v4-pro[1m]","ANTHROPIC_DEFAULT_SONNET_MODEL":"deepseek-v4-pro[1m]","ANTHROPIC_DEFAULT_HAIKU_MODEL":"deepseek-v4-flash","CLAUDE_CODE_DISABLE_NONESSENTIAL_TRAFFIC":"1","CLAUDE_CODE_EFFORT_LEVEL":"max"}}

三个必填参数说明：

  • ANTHROPIC_AUTH_TOKEN - 你的 DeepSeek API Key，没有这个连不上
  • ANTHROPIC_BASE_URL - 接口地址，固定为 api.deepseek.com/anthropic，国内直连
  • 模型映射 - Opus/Sonnet 用 V4 Pro 深度推理，Haiku 用 V4 Flash 快速响应
保存好配置文件后，不需要重启任何东西，下一步直接启动 Claude Code 即可生效。

Step 5：启动 & 验证

5.1 启动 Claude Code

在 PowerShell 中进入你想用的项目目录，运行：

PS> cd $env:USERPROFILE\Desktop\test
PS> claude
Claude Code v2.1.157
Model: deepseek-v4-pro[1m]
Type /help for commands

5.2 首次启动三步引导

第一次运行 claude 命令，会有三步设置：

  1. 选择主题样式：亮色 / 暗色 / 高对比度，随便选一个
  2. 确认安全须知：按 y 回车（Claude Code 可以执行代码，这是正常提示）
  3. 确认目录信任：按 y 回车同意当前目录

5.3 验证模型配置

进入交互界面后，输入以下命令查看当前使用的模型：

> /context
Configured model: deepseek-v4-pro[1m]

如果显示 deepseek-v4-pro[1m]，说明配置成功！可以开始使用了。

也可以用 /model 查看所有可用模型列表：

> /model
Available models:
deepseek-v4-pro[1m] (Opus, Sonnet)
deepseek-v4-flash (Haiku)

5.4 常用快捷键

Ctrl + C 中断当前任务
Ctrl + L 清屏
Ctrl + D 退出 Claude Code
/help 查看所有命令
/clear 清除对话历史

Step 6：常见问题 & 避坑指南

#问题解决方法
1claude 命令找不到重启终端或重启电脑。npm 全局安装的路径需要新会话加载。也可用 npx @anthropic-ai/claude-code 临时启动
2API Key 无效 / 401 错误检查 Key 是否复制完整（无多余空格）。检查 DeepSeek 账户余额。确认 Key 是在 platform.deepseek.com 的 API Keys 页面生成的
3npm install 超时 / 下载慢加 --registry https://registry.npmmirror.com。还可以永久切换：npm config set registry https://registry.npmmirror.com
4settings.json 不生效确认路径：C:\Users\你的用户名\.claude\settings.json。文件名是 settings.json 不是 settings.txt。Windows 默认隐藏扩展名，注意检查
5PowerShell 无法运行脚本运行 Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser 然后按 Y 同意
6Claude Code 启动后一直转圈网络问题。检查 ANTHROPIC_BASE_URL 是否写对。用浏览器打开 https://api.deepseek.com/anthropic 测试能否访问

Bonus：额外技巧

B.1 模型切换工具 CCSwitch

如果你想一键切换不同模型（如切回官方 Claude、换到其他国产模型），推荐安装 CCSwitch 桌面工具。

下载地址：https://github.com/farion1231/cc-switch/releases
下载 CC-Switch-v3.16.0-Windows-Portable.zip，解压即用，无需安装。

配置方式：打开 CCSwitch，点击 + 号，预设中选择 DeepSeek，填入 API Key 即可。

B.2 发送图片到 Claude Code

Claude Code 支持多模态识别，快捷键：Alt + V（Windows） / Command + V（Mac），直接粘贴截图即可让 AI 分析。
