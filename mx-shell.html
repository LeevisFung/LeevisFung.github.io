操作手册总览
本手册基于 Mx-Shell 2026.05.12 直播录屏（约 13 万字）、公开提示词文档（70+ 段落），以及 CSDN/happy-oyster/GitHub 等第三方深度分析综合整理。目标：让你看完就能自己写出一流 AI 视频提示词。
Mx-Shell 创作哲学（三个核心信条）

信条一：像指挥摄影师一样指挥 AI

不要写「关键词列表」，要写「分镜脚本」。AI 需要的是调度指令，不是形容词堆砌。

信条二：写瑕疵 = 写真实

「战损痕迹触目惊心」「各处磨损掉漆」「保留轻微面部瑕疵」——完美的画面就是假的画面。AI 无法凭空产生真实感，你必须明确要求瑕疵。

信条三：抽卡是常态

整部《丧尸清道夫》生成 400 张图片 + 200+ 个视频镜头，每个镜头 2-20 次才能挑出能用的。不是你写得不好，是 AI 本身就有随机性。

五段式提示词模板（核心方法论）
这是 Mx-Shell 所有作品的通用骨架。假面骑士 ×5、丧尸清道夫、合金装备——全用这套结构，只替换变量。 顺序不能乱。
五段式全景
段作用关键词必填/可选
1. 核心主题给 AI 定审美基调3-6 个风格标签必填
2. 人物与基础设定画面里有谁/穿什么/站在哪面部 / 服装 / 场景必填
3. 氛围与画质摄影机型号定画质摄影机 + 镜头 + 色调必填
4. 运镜规则镜头怎么动景别 / 角度 / 呼吸感必填
5. 分镜（时间轴）按秒 or 按镜头展开动作+镜头+特效 三件套必填

段二：人物与基础设定
三行格式：面部 / 服装 / 场景。每行都要有质地、有动态、有瑕疵。
【人物与基础设定】
面部：参照上传图片，五官、脸型、发型百分百还原，杜绝美化。
      面部伤口，纱布，血渍保持一致。刘海覆额，表情全程阴郁。
      变身瞬间，眉心仅轻微蹙起，拒绝昂扬斗志与眼神高光，保持压抑感。
服装：哑光黑色皮质长风衣，金属腰带为哑光质地，腰带核心为暗红色水晶。
场景：末日战场，微风，硝烟弥漫，阴天荒野，灰蓝色调天空。
      天空中划过带有火焰与浓烟的陨石。
关键原则：写质地而不是写颜色
「哑光黑色皮质」信息量是「黑色皮衣」的 3 倍；「生锈金属躯干」>「灰色机器人」
写瑕疵必加：「保留轻微面部瑕疵」「战衣远非平整」「各处磨损掉漆」「关节部位油污」
段三：氛围与画质（全书最有价值的技巧）
Mx-Shell 的独家发现：写具体摄影机型号，比写一百遍「电影感」都有效。AI 训练数据中大量电影元数据都和真实摄影机型号绑定。
【氛围与画质】
视觉基调：变形宽荧幕电影质感。模拟 IMAX 胶片摄影机，
        搭配 Panavision C 系列镜头（焦段 35mm，光圈 f4）。
色彩与影调：低饱和灰蓝主调。暗部信息压缩，保留细节。
          边缘添加轻微柔焦与适度的胶片颗粒感。
风格核心：参考《爱，死亡和机器人》美学风格。
        营造压抑、沉重、实拍的肉身痛苦感。
摄影机 + 镜头速查表（直接抄）：
想要的风格摄影机 + 镜头
史诗感 / 大场面IMAX 胶片摄影机 + Panavision C 系列镜头（35mm，f4）
暗调赛博 / 写实硬核索尼威尼斯电影机 + 佳能 K-35 系列镜头

段四：运镜规则
三行：单镜头 or 多镜头 / 拍摄角度 / 呼吸感。其中「呼吸感」是 Mx-Shell 最核心的技巧。
【运镜规则】
单镜头：一镜到底，无剪辑。
角度：开场仰拍，人物半身左侧面 30 度，聚焦腰部以上部位。
     开始变身后镜头以极缓速度环绕至正面平视上半身。
     变身完成后环绕至人物右侧面俯拍。
呼吸感：手持拍摄，全程保持极其轻微的、如呼吸般的镜头浮动，增强临场感。
「呼吸感」为什么如此重要？
AI 默认输出绝对静止的 CG 画面。加上「呼吸般的浮动」，画面立刻像真人手持摄影机拍摄。
一句话改变质感。Mx-Shell 每个视频提示词都写这句话。
运镜词汇库：
类别词汇
景别极近距离特写 / 特写 / 近景 / 中景 / 全景 / 大全景
角度仰拍 / 俯拍 / 平视 / 过肩镜头 / 高空垂直俯拍
运动极缓向前推进 / 环绕 / 跟随运动拍摄 / 固定机位 / FPV 模式
节奏极缓 / 匀速 / 突然 / 应激性 0.1 秒微颤 / 极慢动作
段五：分镜（时间轴）
两种写法按需选择——单镜头用「按秒切片」，多镜头用「按镜头切片」。

分镜写法详解
写法 A：按秒切片（适合单镜头一镜到底）
每段「动作 + 镜头 + 特效」三件套，可选择加「声音」「面部」。
【分镜时间轴】

0-3秒 · 凝视
动作：主角略微低头，目光锁定腰带，脊背紧绷。
     右手缓慢抬起，握住腰带主体。
镜头：极缓向前推进，捕捉身体因呼吸产生的微幅起伏。
特效：整颗眼球骤然亮起白金色光芒，眼眶周围皲裂。
     眼睛亮起的炫光使画面镜头产生光斑。

3-6秒 · 启动
声音：低语「HENSHIN」
动作：手掌用力按压腰带核心，水晶核心本体被挤压而开裂，
     红光伴随晶体的裂纹亮起。
特效：金属机械装置被暴力唤醒，核心主体沿中线开裂。
     空气向内塌陷，产生畸变弧光。
镜头：低频嗡鸣由远及近，镜头随之一同产生
     应激性的 0.1 秒微颤。

6-9秒 · 撕裂
动作：身体开始异化，甲壳从脊椎突出...
特效：电光火石间，战衣从破裂肌肉中长出...
镜头：镜头极缓环绕，捕捉变身全貌...
写法 B：按镜头切片（适合多镜头叙事）
每个分镜四件套：景别 + 构图 + 运镜手法 + 画面内容。这是《丧尸清道夫》实际使用的方法。
【丧尸清道夫 分镜示例】

分镜一：
景别：中景，拍摄机器人腿部以上位置
构图：机器人背影作为前景居中位于画面中心，
     背景左侧落地窗，右侧吧台
运镜手法：跟随运动拍摄
画面内容：机器人背对镜头向前方移动，
         手舞足蹈进入别墅休闲区，潇洒转身作为舞蹈 END 动作，
         led 显示器保持绿色微笑表情。
         片刻后画外传来玻璃酒瓶被打碎的声音，
         机器人受到惊吓，下意识向画面左侧后撤半步，
         led 显示器瞬间切换至黄色害怕表情。

分镜二：
景别：全景，平视拍摄吧台全貌
构图：空镜平视拍摄吧台全貌，
     画面下方三分之一处为吧台和吧台椅子，
     上方为墙面酒柜
运镜手法：镜头缓慢前推，渲染氛围
画面内容：吧台后方传来悉悉簌簌的声音，
         像是有什么东西躲在吧台下翻找

分镜三：
景别：特写
构图：机器人正面特写，位于画面中心
运镜手法：固定机位，镜头跟随机器人微颤
画面内容：机器人摆出防御姿态，双手交叉于胸前，
         关节微微颤抖，LED 保持黄色害怕表情
选择规则：
变身、武器充能、单一场景 → 用 写法 A（按秒）
多个场景切换、有故事线、有对话逻辑 → 用 写法 B（按镜头）
4
10 个立刻有效的实战技巧

不要给 AI 参考图（除非画质够高）

AI 垫图时容易复现参考图的画风（包括 CG 感、动漫感），而不是参考它的设计。Mx-Shell 的假面骑士系列全部靠文字描述生成。例外：参考图是 3D 渲染或高画质实拍。

写摄影机型号代替「电影感」

AI 训练数据中的电影元数据和具体摄影机型号绑定。「IMAX 胶片摄影机 + Panavision C 系列镜头 35mm f4」比「电影感」精确一百倍。

描述瑕疵 = 描述真实

所有提示词都加：「保留轻微面部瑕疵」「战损痕迹触目惊心」「各处磨损掉漆」「关节部位油污」。完美的画面 = 假的画面。

给镜头加「呼吸感」

手持拍摄，全程保持极其轻微的、如呼吸般的镜头浮动，增强临场感。一句话消灭 CG 僵直感。

接受抽卡是常态

不要追求一次成型。每个镜头 2-20 次才能挑出能用的。Mx-Shell 的哲学：「我可以同时抽着两个场景的卡，然后编写着第三个场景的词。」

版权违规 → 删字眼换近义词

Seedance 2.0 对版权敏感。钢铁侠 → 原子朋克未来主义复古风；HENSHIN → 自创音节。遇到拦截不要硬刚，找近义词。

长视频 = 短片段 + 剪辑拼接

30 秒一镜到底很难抽。Mx-Shell 的《合金装备》打斗段就是分两段生成 + 后期剪辑。不要追求一口气生成。

同一色调，前后镜头才能拼

最大剪辑坑：A 镜头偏暖橙，B 镜头偏冷蓝，剪在一起色差爆炸。在场景图阶段就把色调定死，所有视频提示词的「色彩与影调」段统一。

没必要做 4K，720p 够了

Mx-Shell 的原话：「我的视频基本都是 720p。」国内平台会再压缩。省下来的算力多抽几次卡更重要。

不一定要用人脸

头盔/面具/机器人更容易。AI 最难保持人脸的跨镜头一致性。这是《丧尸清道夫》选机器人主角的深层原因——LED 像素脸比真人脸稳定一万倍。

5
常见问题调试指南
问题根因解决方案
画面不稳定 / 闪烁 动作描述太多 把「walks, fights, jumps, escapes」改成「walks slowly, notices movement, reacts with one controlled strike」。遵守三步动作链原则。
主角面部漂移 参考图约束不够 加一行：「Use the reference image ONLY to preserve facial identity, hairstyle, body proportion. Do not copy unrelated clothing, background, or pose.」
风格不统一 风格词互相冲突 如果同时写 realistic + anime + cyberpunk + documentary，AI 会在不同帧间跳风格。只保留一个主风格，其他用 lighting 和 texture 描述。
丧尸动作太流畅 没写动作方式 写成：「moves unevenly from behind an abandoned car, with stiff shoulders, slow unstable steps, and unnatural head movement」——用具体运动方式代替形容词。
画面「AI 味」太重 大词太多，具体视觉描述太少 把「ultra detailed, masterpiece, epic, dramatic」全部删掉，换成可执行的物理细节：湿沥青反射灯光、沙尘漫过路牌、远处建筑因热浪扭曲。
动作链规则：5-8 秒视频只能容纳三步
这是最常见的失败原因——试图在一个短片段里塞太多动作。
# 错误：动作密度过高
She runs, fights zombies, jumps over cars, shoots,
explodes everything, escapes from the city.
→ 主角瞬移、丧尸数量失控、镜头乱切

# 正确：三步动作链
第一步：在废弃车辆之间缓慢行走
第二步：举起手电筒照向移动的影子
第三步：以控制性的近距离动作击退靠近的丧尸
→ 一个镜头内完成，节奏清晰

创作前自检清单
  • 核心主题是否包含 3-6 个风格标签？
  • 人物设定是否写了「面部 / 服装 / 场景」三行？
  • 服装是否写了质地（哑光/皮质/金属）而非仅颜色？
  • 是否加了「保留轻微面部瑕疵」或战损描述？
  • 是否用了具体摄影机型号而非「电影感」？
  • 是否加了「呼吸感」一行？
  • 分镜是否按秒或按镜头拆分，每段有「动作+镜头+特效」？
  • 动作是否遵守三步动作链原则（5-8 秒只写三步）？
  • 是否加了负面约束列表？
  • 是否做好了抽卡 2-20 次的心理准备？
